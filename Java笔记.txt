1.成员变量不可以使用访问权限修饰符（如public）,因为只可以在方法中使用。
2.类变量和成员变量的理解。类变量加static修饰符，一个类的多个对象之间可以
相互使用。成员变量不可以，类对象的成员变量值相互独立。
3.开闭原则：对扩展是开放的，对修改时关闭的。（多使用配置文件）
4.i++和i--    先赋值后运算  先运算后赋值

DOS下操作：
	cd  ...
	javac 文件名.java  //编译程序
	java 文件名    //运行

字符编码:
Unicode:采用16位的字符编码，几乎可以表示世界上所有的语言文字字符。
AscII码:采用8位的字符编码，只能表示英文字符，不能表示所有字符。
Utf-8：最广泛的编码方式。变长编码方式，可以用1-6个字节来表示所有字符。
GB2321：针对汉字的编码方式。


一。java基础：
1.注释
	单行注释 //
	多行注释  /*.....*/
	文档注释 /**.....*/  作用：属于多行注释，可以使用文档生成器（javadoc命令）读取java程序并提取其中的文档注释，生成所编程序的HTML文件形式的API文档，作为该程序的java帮助文档。

2.基本语法成分：标识符，关键字，分隔符，运算符，常量
 分隔符：\t制表符 \r回车 \n换行...
 关键字：已经被赋予特定意义的单词符号。不能作为类名，方法名，变量名等使用。 如：static,int
 标识符：凡是需要自己命名的地方。如类名，方法名等。  由数字，字母，下划线和美元符号组成。


3.数据类型：基本数据类型和引用数据类型。
基本数据类型：数值类型（整形和浮点型），字符类型，布尔类型。
引用数据类型：类，接口，数组

4.变量。
作用域：一对{}之间。
类别：成员变量和局部变量
注意：！！！用static说明的变量叫类变量，也称静态变量，用static说明的方法叫类方法，类变量和类方法不需要通过创建对象来访问，可以通过类来直接访问。但类方法不允许重载。每个对象对类变量的修改都会影响其他实例对象。
  ！！！用final修饰的成员变量被称为常量，常量初始化后不能被修改。

二。java面向对象：
类的成员：属性，方法，构造器（构造方法）
